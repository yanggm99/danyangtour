# 단양 맞춤형 관광 큐레이션 플랫폼 설치 가이드

이 문서는 제공된 파일들을 그누보드 환경에 적용하는 방법을 설명합니다.

## 1. 사전 준비

1. **그누보드 5**가 설치되어 있어야 합니다.
2. `tour` 라는 ID를 가진 게시판을 생성해주세요. (관리자 > 게시판관리 > 게시판생성)
   - TABLE: `g5_write_tour` 가 생성됩니다.
3. 사용할 테마의 폴더 위치를 확인하세요. (예: `theme/basic`)

## 2. 데이터베이스 확장 (필수)

제공된 `add_fields.sql` 파일을 사용하여 게시판 테이블에 필터 저장용 컬럼을 추가해야 합니다.

1. phpMyAdmin 또는 DB 관리 도구에 접속합니다.
2. `add_fields.sql` 파일의 내용을 복사합니다.
3. SQL 실행 창에 붙여넣고 실행합니다.
   - **주의**: 만약 게시판 테이블명이 `g5_write_tour`가 아니라면, SQL 파일 내의 테이블명을 실제 테이블명으로 변경해야 합니다.

## 3. 파일 업로드

다음 파일들을 해당 경로에 업로드하세요.

| 파일명                | 경로                                          | 설명                             |
| --------------------- | --------------------------------------------- | -------------------------------- |
| **filter_config.php** | `/theme/basic/filter_config.php`              | 50개 필터 옵션 정의 설정 파일    |
| **write.skin.php**    | `/theme/basic/skin/board/tour/write.skin.php` | 글쓰기/수정 스킨 (체크박스 포함) |
| **view.skin.php**     | `/theme/basic/skin/board/tour/view.skin.php`  | 글보기 스킨 (태그 출력)          |
| **list.skin.php**     | `/theme/basic/skin/board/tour/list.skin.php`  | 목록 스킨 (필터 검색바 포함)     |

> **폴더가 없다면?**
> `theme/basic/skin/board` 안에 `tour` 폴더를 새로 만들고 스킨 파일들을 넣으세요.
> 그 후 관리자 페이지 > 게시판 관리 > 'tour' 게시판 수정 > 스킨 설정을 `(테마) tour`로 변경해야 합니다.

## 4. 확인 및 테스트

1. **관리자 로그인** 후 게시판 글쓰기 페이지로 이동합니다.
2. **"상세 옵션 선택"** 영역에 체크박스들이 보이는지 확인합니다.
3. 몇 가지 옵션을 체크하고 글을 저장합니다.
4. 작성된 글을 볼 때, 체크한 옵션이 **태그(#)** 형태로 나오는지 확인합니다.
5. 목록 화면 상단에 **필터 검색바**가 나오는지 확인합니다.

---

**문의사항**: 설치 중 에러가 발생하거나 테이블명이 다르다면 알려주세요.
